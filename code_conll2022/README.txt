To run the experiments you need to:

1. Install Transformers 4.20
2. Place the train.dev, dev.tsv and test.tsv files from cometa_dataset_v1 (and
VUAM) in the path of the cometa_dataset.py script.
3. Generate the scripts using generate_scripts.py and low_generate_scripts.py
4. Run (you can just run the best models using the hyperparameters specified in
the paper.
